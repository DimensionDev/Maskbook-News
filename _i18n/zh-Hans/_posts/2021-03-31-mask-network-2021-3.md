**- DeFi**

**⛓️** [**Mask Network x NEAR Protocol｜深度合作 & 联合赏金**](http://mp.weixin.qq.com/s?__biz=MzU4OTkwNDYzMw==&mid=2247488169&idx=1&sn=fc31ad4b3c7fdc1a552fd9d88602a0fb&chksm=fdc73d3fcab0b4290583680efe755ec85f4711847f957abb40bdee3dc046dd38a305359e4586&scene=21#wechat_redirect)

Mask Network 与 PoS 底层公链 NEAR Protocol 达成深度合作，在技术以及应用等多层面进行交流。Mask Network 将有望接入其跨链工具彩虹桥（Rainbow Bridge）、EVM （以太坊虚拟机）、转账工具 NEAR Drop 等功能以完成 Mask Network 与 NEAR Protocol 的集成。

合作以发布联合赏金的方式启动，首个赏金的任务是将 NEAR Wallet 率先接入到 Mask Network 插件中，为未来进一步的合作奠定坚实基础。

**🤝** [**Mask Network x Convergence｜预计实现 WSTs 的 ITO 发行**](http://mp.weixin.qq.com/s?__biz=MzU4OTkwNDYzMw==&mid=2247488262&idx=1&sn=c0cfb4ae61d46ae87db0812836716a3a&chksm=fdc73c90cab0b5866053abc935731c033c17c0ccfa6bb0bdd6c135e290ac4c6790a27ea485a1&scene=21#wechat_redirect)

Mask Network 投资并与 DeFi 可互换资产协议 Convergence 合作，同时，Mask Network 创始人以及 CEO Suji Yan 也将成为 Convergence 的顾问之一。

Convergence 上的项目计划通过 Mask Network 特有的 ITO 功能发行 Wrapped Security Tokens （WSTs），且后续允许用户在 Convergence 的 AMM 池中进行交易。

**🙌** [**Mask Network x Polygon｜探索更多玩法**](http://mp.weixin.qq.com/s?__biz=MzU4OTkwNDYzMw==&mid=2247488276&idx=1&sn=e4ed50071afe4d9dfd9410a585af8958&chksm=fdc73c82cab0b594a6bab64b0c47f2efa80bf6e5682f4cdf7993732cad20c9d6b2a6992529b5&scene=21#wechat_redirect)

Mask Network 与 Layer 2 解决方案聚合器 Polygon（原 Matic Network） 合作，双方针对以太坊目前出现的拥堵以及高矿工费问题展开技术交流协作。

Mask Network 将率先接入 Polygon 生态项目之一的去中心化交易所 QuickSwap，并继续通过发布赏金的形式逐步完成与 Polygon 的集成。
